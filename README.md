## 🛠️ Usage

This repository serves as a template for GitHub's special `.github` directory. To use these templates in your project:

1. Copy the relevant files to your project's `.github/` directory
2. Customize the templates to fit your project's needs
3. Commit and push the changes

### For Organizations

If you're setting this up for an organization:

1. Create a repository named `.github` in your organization
2. Copy these templates to that repository
3. The templates will automatically apply to all repositories in the organization
